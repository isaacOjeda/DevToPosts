# Facturación

## Cómo generar una factura
1. Ve al módulo **Facturación** en el menú principal.
2. Selecciona el cliente desde el catálogo.
3. Agrega los productos o servicios con sus cantidades.
4. Revisa el subtotal, impuestos y total.
5. Haz clic en **Generar Factura**. El sistema asignará el folio automáticamente.

## Formatos de factura disponibles
- **PDF**: para envío por correo o impresión.
- **XML (CFDI)**: requerido para efectos fiscales. Se genera automáticamente junto con el PDF.
- **CSV**: exportación de datos para sistemas externos.

## Fechas de corte
- El cierre mensual es el último día hábil de cada mes.
- Las facturas generadas después de la fecha de corte se registran en el mes siguiente.
- No se pueden modificar facturas de meses anteriores ya cerrados.

## Cancelación de facturas
Solo los usuarios con rol Administrador o Supervisor pueden cancelar facturas. Para cancelar:
1. Busca la factura por folio o cliente.
2. Haz clic en **Cancelar** y selecciona el motivo.
3. El sistema enviará automáticamente la cancelación al SAT.

## Problemas comunes
- **"No se puede timbrar"**: verifica que el certificado digital (CSD) esté vigente. Contacta a soporte@empresa.com si el problema persiste.
- **Factura duplicada**: no regeneres la factura. Contacta a soporte para revisar el folio.
